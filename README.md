# 🍫 ChocoInst

**ChocoInst** — это графическая программа для установки пакетов через **Chocolatey** с удобным интерфейсом и переключением тем.

## ✨ **Возможности**

- 📦 **Установка любых пакетов** из Chocolatey
- 🎨 **5 тем оформления** на выбор:
  - 🌑 Тёмная
  - ☀️ Светлая
  - 🔵 Синяя
  - 💚 Матрица
  - ⚪ **Бело-голубая** (любимая тема Марка)
- 🚀 **Прогресс установки** с выводом команд
- 🔍 **Автопоиск Chocolatey** в системе
- ⚡ **Запрос прав администратора** (автоматически)
- 📋 **Подробный вывод** в реальном времени
